# NEX.AI Branding

## Name
**NEX.AI**

## Tagline
AI Native Data Platform

## Full Description
NEX.AI is a comprehensive, AI-powered data platform that serves as a one-stop solution for all data-related needs in large organizations.

## Logo Concept
- Primary: Database icon with gradient (blue → purple → pink)
- Modern, clean, professional, cutting-edge

## Color Palette

### Dark Mode (Primary)
- **Background**: `#0a0e27` - Deep space blue
- **Surface**: `#141829` - Dark navy
- **Card**: `#1a1f3a` - Rich dark blue
- **Border**: `#2a2f4a` - Muted border
- **Text**: `#e2e8f0` - Light slate
- **Muted Text**: `#94a3b8` - Slate gray

### Accent Colors
- **Blue**: `#3b82f6` - Vibrant blue
- **Purple**: `#a855f7` - Electric purple
- **Pink**: `#ec4899` - Hot pink
- **Cyan**: `#06b6d4` - Bright cyan

### Gradients
- Primary: Blue → Purple → Pink
- Secondary: Cyan → Blue
- Success: Green → Cyan
